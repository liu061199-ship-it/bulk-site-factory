# Templates

The active templates are implemented in `src/app/page.tsx` so the project stays small and easy to run.

Supported `sites/sites.json` template values:

- `guide`: evergreen guide layout
- `news`: update/news layout
- `hub`: resource-center layout
- `tips`: short-tips layout
- `review`: review/checklist layout

Keep template selection, keywords, content focus, audience, and content pillars in `sites/sites.json`. Do not hard-code site-specific content in page components.
